Set Resource Command

setResource command allows for the contents of a resource in the classpath to be set as the value of a variable and appended as the body of the element where setResource command is used. The resource path is specified via the path command. The resource path given should be relative to the spec that uses the resource.

Given a file SetResourceSpecSample.xml exists in the same location as this spec, an empty pre instrumented with setResource="#xmlContent" and path="SetResourceSpecSample.xml" should display the content of SetResourceSpecSample.xml below: 

<?xml version="1.0" encoding="UTF-8"?>
<sample>
    <sub-element sample-attribute="sample-attribute-value">
        <sample-empty-element/>
        <sample-element-with-body>
            Sample body
        </sample-element-with-body>
        <sample-mixed-element another-sample-attribute="another-sample-attribute-value">
            Another sample body
        </sample-mixed-element>
    </sub-element>
</sample>

And an empty pre instrumented with echo="#xmlContent" should display the same content below: 

<?xml version="1.0" encoding="UTF-8"?>
<sample>
    <sub-element sample-attribute="sample-attribute-value">
        <sample-empty-element/>
        <sample-element-with-body>
            Sample body
        </sample-element-with-body>
        <sample-mixed-element another-sample-attribute="another-sample-attribute-value">
            Another sample body
        </sample-mixed-element>
    </sub-element>
</sample>
