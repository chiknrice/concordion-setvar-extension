Tooltip Command

tooltip command sets the evaluation result of the expression specified as the title attribute of the instrumented element.

Given a content "The tooltip content" wrapped with span which is instrumented with set="#tooltipContent"

Then the words "i have a tooltip" wrapped with a span instrumented with tooltip="#tooltipContent" should display "The tooltip content" when the mouse pointer hovers.

tooltip can also be used in tables as below:

Given a content "the tooltip" wrapped with span which is instrumented with set="#tooltipContent"

Then the table's th instrumented with <table c:execute="#right = echo(#left)">, <th c:set="#left"></th>, and <th tooltip="#right + ' with ' + #tooltipContent"></th> should set a title for all corresponding td
